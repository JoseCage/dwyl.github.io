What do we believe in?

Fully Tested Code

We're proud of the quality of our code. Having automated tests means that when we add to that code we know we're not creating a problem somewhere else.

Simplicity

We keep it simple and intuitive, with only the features you need and a tried and tested user journey.

Open Source

We're big advocates of developing our code out in the open, allowing us to draw upon a global network of expert contributors.

Ethics Driven

We people who are doing amazing things and having a positive effect on the world. If that's you, we'd love to hear from you!
